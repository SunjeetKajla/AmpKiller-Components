ESP32-CAM

Camera Module

OV2640 camera module for real-time video streaming and visual assessment of collapsed structure interiors.

Technical Specifications

  • 2MP OV2640 camera sensor
  • WiFi Direct capability
  • Real-time video streaming
  • JPEG compression
  • Flash LED support

System Role

Provides live visual feed of collapsed structure interior for remote assessment and survivor detection.

Arduino Nano

Secondary Microcontroller

Compact microcontroller for real-time control tasks and legacy module interfacing.

Technical Specifications

  • ATmega328P processor
  • 14 digital I/O pins
  • 8 analog inputs
  • 16MHz clock speed
  • USB connectivity

System Role

Handles real-time sensor interfacing and legacy module communication for seamless integration.

ESP32

Main Controller

Dual-core processor with WiFi and Bluetooth capabilities serving as the central processing unit.

Technical Specifications

  • Dual-core Xtensa LX6 CPU
  • WiFi 802.11 b/g/n
  • Bluetooth Classic & BLE
  • 520KB SRAM
  • Multiple communication protocols

System Role

Central processing unit for sensor data fusion and wireless communication coordination.

MQ-2 Gas Sensor

Gas Detection Module

Multi-gas sensor for hazardous gas detection in potentially dangerous environments.

Technical Specifications

  • LPG, CO, smoke detection
  • 10-10000ppm detection range
  • Analog voltage output
  • Fast response time
  • Low power consumption

System Role

Detects dangerous gas leaks and smoke in collapsed structures to ensure operator safety.
